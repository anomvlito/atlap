# Documentación de Atlap

Bienvenido a la documentación del proyecto Atlap. Aquí encontrarás guías para iniciar el proyecto, la arquitectura, consideraciones de seguridad y cómo contribuir.

Contenido principal:

- `getting-started.md` — Pasos para ejecutar el proyecto en desarrollo.
- `architecture.md` — Diseño de alto nivel y organización del código.
- `security.md` — Riesgos, mitigaciones y recomendaciones de despliegue seguro.
- `api.md` — Propuesta de endpoints y contrato API server-side (recomendado).
- `contributing.md` — Flujo de trabajo para contribuciones y PRs.

Empieza por `getting-started.md` para levantar el proyecto localmente.
